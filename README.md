# UNF Computer Science — Coursework

Projects from my B.S. in Computer Science at the **University of North Florida** (2018–2020). Each folder holds the work for one course, spanning compilers, systems programming, networking, algorithms, data structures, and web development — in Python, Java, C, C++, and PHP.

## Projects by course

### Compilers — [`Compilers/`](Compilers)
A full multi-stage **C-Minus compiler**, built as a five-project series: lexical analyzer → parser → semantic analyzer → intermediate representation → code generation. Validated against a standardized test-case suite.
**Tech:** Python, lex/yacc

### Computer Networks — [`Computer Networks/`](Computer%20Networks)
A TCP **socket server/client** implemented two ways — an **iterative server** and a **multithreaded concurrent server** — demonstrating the tradeoffs between sequential and parallel request handling.
**Tech:** Java (sockets, threads)

### Operating Systems — [`Operating Systems/`](Operating%20Systems)
Three classic systems problems: **Banker's algorithm** for deadlock avoidance, a **producer–consumer** solution with thread synchronization, and a **Unix-style command interpreter** in C.
**Tech:** Java, C

### Intro to AI — [`Intro to AI/`](Intro%20to%20AI)
Implementations of **A\***, **ARA\*** (Anytime Repairing A\*), and **bounded A\*** search algorithms.
**Tech:** Python

### Data Structures — [`Data Structures/`](Data%20Structures)
Data-structure implementations, including a singly-linked-list-backed **MP3 playlist manager** and array/merge utilities.
**Tech:** Java

### Internet Programming — [`Internet Programming/`](Internet%20Programming)
**"myMovies Xpress!"** — a session-authenticated movie shopping-cart web app (PHP + MySQL, OMDb API) with login, keyword search, AJAX detail modals, and a server-side cart.
**Tech:** PHP, MySQL, JavaScript, HTML/CSS
